# news-feed

React news feed in which you can search and check a news feed.

### Frontend

- React.js v18
- React-router-dom v6
- Redux-toolkit
- Typescript
- Vite(boost development experience)
- Scss(Sass)

### Backend

- Node.js
- Express
- Typescript
- News API([https://newsapi.org/](https://newsapi.org/))

### Deployed on

- Vercel (Frontend)
- Heroku(Backend)
